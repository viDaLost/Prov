# Сайт-приглашение «Маша + Даня»

Готовый статический сайт: можно открыть `index.html` прямо на телефоне или загрузить папку на GitHub Pages / любой хостинг.

## Что сделано

- 3 скриншота аккуратно соединены в один длинный фон без заметных стыков: `assets/invitation-background.webp` и JPG-фолбэк.
- Кнопки наложены на нужные места и работают:
  - маршрут до регистрации;
  - маршрут до банкета;
  - воспоминания / выбор и отправка файлов через системное меню телефона;
  - анкета гостя с сохранением ответа;
  - добавление события в календарь.
- Добавлены мягкие анимации, плавающие сердечки, прелоадер, кнопка «наверх», живой таймер обратного отсчёта.
- Всё адаптировано под телефон.

## Как изменить данные

Откройте `config.js` и замените адреса, ссылку на Telegram-группу или endpoint анкеты.

```js
window.WEDDING_CONFIG = {
  weddingDate: '2026-08-26T16:00:00+03:00',
  locations: {
    registration: { address: '...' },
    banquet: { address: '...' }
  },
  links: {
    memories: 'https://t.me/...',
    rsvpEndpoint: '',
    rsvpEmail: ''
  }
};
```

Если `rsvpEndpoint` пустой, анкета сохраняется локально в браузере и её можно скопировать.
