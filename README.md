# Сайт-визитка программиста

Статический премиальный сайт-визитка, собранный на чистых HTML/CSS/JS без зависимостей и сборщика.

## Что использовано из прикреплённых архивов

- `make-interfaces-feel-better`: концентрические радиусы, точные hover/press-состояния, interruptible CSS transitions, stagger reveal, `text-wrap: balance/pretty`, табличные цифры, аккуратные поверхности.
- `ui-ux-pro-max-skill`: дизайн-токены, семантические CSS-переменные, компонентный подход, сетки, адаптивные layouts, брендовая структура.
- `impeccable`: системная полировка интерфейса: типографика, layout-аудит, визуальная иерархия, reduced motion, доступность.
- `magic-mcp`: подход natural-language → polished component system: интерфейс собран как набор переиспользуемых UI-блоков.

## Файлы

- `index.html` — структура сайта.
- `styles.css` — дизайн-система, адаптив, анимации, светлая/тёмная тема.
- `app.js` — интерактив: тема, командная панель, scroll progress, reveal, форма, vCard.
- `assets/favicon.svg` — векторная иконка.

## Как запустить

Откройте `index.html` в браузере или разверните папку на GitHub Pages / Netlify / Vercel / любом статическом хостинге.

## Что заменить под себя

В `index.html` замените имя, специализацию, тексты кейсов и email. В `app.js` обновите объект `profile`, чтобы корректно работали копирование email и vCard.
