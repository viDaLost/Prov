# Свадебное приглашение — Маша + Даня

Готовый адаптивный сайт из трёх новых изображений.

## Что внутри

- `index.html` — основной сайт.
- `style.css` — адаптивная вёрстка, анимации, точные кликабельные зоны.
- `app.js` — таймер, календарь, кнопки, анкета, модальные окна.
- `config.js` — ссылки и настройки, которые можно менять без правки сайта.
- `assets/` — изображения в PNG + WebP 2x для максимальной чёткости на телефонах.

## Что работает

- Сердечко в календаре анимировано. Нажатие скачивает `.ics` файл для добавления свадьбы в календарь.
- Две кнопки маршрута открывают Яндекс.Карты по адресам.
- Кнопка «Открыть чат» готова к Telegram-ссылке через `config.js`.
- Кнопка «Заполнить анкету» открывает встроенную анкету, пока внешняя ссылка не указана.
- Таймер внизу живой и считает до 26.08.2026.
- Сайт адаптивен: телефон, планшет, компьютер.

## Как заменить ссылки

Откройте `config.js` и заполните:

```js
memoriesUrl: 'https://t.me/+ВАША_ССЫЛКА',
formUrl: 'https://forms.gle/ВАША_АНКЕТА',
```

Если `formUrl` пустой, будет работать встроенная анкета на сайте.
